# College Bus Schedule and Route Map Application

## 📌 Overview
The College Bus Schedule and Route Map Application is a mobile application developed using Flutter and Firebase to help students and staff access bus schedules, view routes, and receive delay notifications.

## ✨ Features
- Live bus tracking
- Bus schedules and route information
- Delay notifications
- User, Driver, and Admin modules
- Firebase backend integration

## 🛠 Technologies Used
- Flutter
- Dart
- Firebase
- Cloud Firestore
